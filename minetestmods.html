Minetest Mods

In my spare time, I have created quite a few mods for Minetest, because it's just so cool.
Here is a list of the mods I ended up publishing to GitHub.

  • effects, which adds a status effect API.
  • silicons, which is a -cons with a floating point state.
  • lolchat, which adds various cool chat commands, but has been in a bit of a... broken state recently.
  • rmod, including conveyors, grates, and a block-based programming language.
  • digimagic, enabling luacontrollers to modify the world around them.
